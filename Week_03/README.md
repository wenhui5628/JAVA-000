### 1）Netty概览
    定义：网络应用开发框架  
    特点：1、异步 2、事件驱动 3、基于NIO
    适用场景：1、服务端 2、客户端 3、TCP/UDP
    需要了解的概念：
    什么是事件驱动？
    传统面向接口编程是以接口为媒介，实现调用接口者和接口实现者之间的解耦，但是这种解耦程度不是很高，如果接口发生变化，双方代码都需要变动，而事件驱动则是调用者和被调用者互相不知道对方，两者只和中     间消息队列耦合。
